## Class: AppModel
A base model.
All the app models must inherit from this.
The general code of the models goes here.

Example :

```js
define([
    'core/models/AppModel',
], function(AppModel) {
    'use strict';
    return AppModel.extend({
        route: '/deals',
    });
});
```

**route**: `String` , The API route
